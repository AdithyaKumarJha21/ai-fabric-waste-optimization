# Dataset

This folder contains the data generation code and sample dataset for the fabric waste prediction system.

## Purpose

- Generate realistic fabric cutting records.
- Store the generated CSV dataset.
- Provide training data for the Random Forest model.

## Files

- `generate_dataset.py` - creates the fabric waste dataset.
- `fabric_waste_dataset.csv` - generated dataset used for model training and testing.

## Notes

- Keep dataset creation code here only.
- Do not place layout calculations or machine learning training code in this folder.
