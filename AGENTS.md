# Project Rules

Project Name:
AI-Powered Fabric Waste Prediction & Cutting Optimization System

Purpose:
Build an intelligent fabric cutting optimization system for factories and tailors.

Architecture:
- dataset/ -> dataset generation
- logic/ -> mathematical calculations
- model/ -> AI prediction models
- backend/ -> Flask/FastAPI backend
- frontend/ -> UI application

Important Rules:
- Do NOT mix logic calculations with ML code.
- Layout calculations must stay inside logic/.
- Waste prediction belongs inside model/.
- Keep code modular.
- Use beginner-friendly readable code.
- Add comments for every major step.
- Avoid overengineering.
- First build working system using Random Forest.
- SNN integration will happen later.
