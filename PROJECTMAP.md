# Project Map

## System Purpose
Predict fabric waste and optimize cutting layouts.

## Main Modules

### dataset/
Generates realistic textile cutting datasets.

### logic/
Handles:
- rows calculation
- columns calculation
- pieces calculation
- layout guidance

### model/
Handles:
- waste prediction
- efficiency prediction

### backend/
Handles:
- API endpoints
- model inference
- integration

### frontend/
Handles:
- user input forms
- result display
- layout visualization

## Final Outputs
- Pieces Possible
- Waste %
- Utilization %
- Efficiency Level
- Suggestions
- Layout Guidance
